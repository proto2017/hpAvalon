# Hpavalon

## 介绍
使用webpack2构建，基于avalon2，pc端后台网站类型快速开发的脚手架
支持ie8+

## 功能：
* 集中式为主，扁平化为辅的开发目录结构 doing
* 支持es6语法和模块化 todo
* 支持less todo
* 样式自动添加前缀 todo
* 支持前端接口代理 todo
* 支持mock数据 todo
* 编辑自动刷新 todo
* 支持压缩构建 todo
* 支持单元测试 todo
* git使用推送自动发布环境 todo
* 使用符合fetch的标准库进行异步请求 todo
* 使用redux进行数据流管理 todo
* 使用underscore进行数据处理 todo
